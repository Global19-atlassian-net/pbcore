pbcore¶

The pbcore package provides Python modules for processing PacBio data files and building PacBio bioinformatics applications.

Library API documentation¶

pbcore.io: Classes for reading/writing PacBio data formats and essential common data formats

pbcore.model: Common base classes

pbcore.util: Utilities for building bioinformatics applications

pbcore.data: Small bundled data files that are handy for testing and debugging
